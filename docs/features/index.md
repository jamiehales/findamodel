# Features

FindAModel provides several pages for managing and working with your 3D model library.

| Page | Description |
|------|-------------|
| [Explorer](explorer) | Browse directories, edit per-directory config, trigger indexing |
| [Models](models) | Search and filter all indexed models |
| [Printing plates](printing-plates) | Build and manage printing plates with hull-based layout |
| [Settings](settings) | Global application settings |
| [Auto-support V2 optimization plan](autosupport-v2-optimization-implementation) | Detailed implementation plan for adaptive-resolution method 2 performance improvements |
